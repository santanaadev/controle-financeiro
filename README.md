# Controle de Gastos em Java

Projeto em Java desenvolvido com foco em **aprendizado sólido de Programação Orientada a Objetos** e boas práticas de organização de código, simulando um sistema real de controle financeiro.

Este repositório representa meu **projeto principal de evolução em Java**, sendo expandido gradualmente conforme avanço nos estudos.

---

## Objetivo

- Consolidar fundamentos da linguagem Java
- Aplicar conceitos de POO em um projeto realista
- Desenvolver raciocínio de arquitetura e separação de camadas
- Criar uma base sólida para futuras integrações (banco de dados e automações)

---

## Conceitos Trabalhados

- Programação Orientada a Objetos (POO)
- Encapsulamento e responsabilidade das classes
- Separação em camadas (model, repository, service)
- Organização e legibilidade de código
- Evolução incremental do software

---

## Estrutura do Projeto

src/
├── model # Entidades do domínio (ex: Gasto)
├── repository # Camada de acesso aos dados
├── service # Regras de negócio
└── main # Execução da aplicação
A estrutura foi pensada para facilitar manutenção, testes e futuras integrações.

---

## Funcionalidades Atuais
